# AI Long-Term Memory

Этот каталог предназначен для хранения долговременной памяти и контекста проекта для ИИ-агента.
Здесь будут храниться архитектурные решения, паттерны, история изменений и текущее состояние, чтобы не терять контекст между сессиями.

## Структура
- `PROJECT_CONTEXT.md` - Полный контекст проекта, цели и текущий статус.
- `ARCHITECTURAL_DECISIONS.md` - Журнал принятых технических решений (ADR).
- `TROUBLESHOOTING.md` - База знаний по решению проблем (например, BIOS/Docker).
