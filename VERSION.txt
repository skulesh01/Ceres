Ceres Infrastructure v1.0

Project: Ceres - Modular Corporate Infrastructure
Date: 04.12.2025
Status: Production Ready

Features:
- 16 microservices
- Single Docker Compose configuration
- Unified authentication (Keycloak SSO/OIDC)
- Persistent data storage (Docker volumes)
- Automatic HTTPS (Traefik + Let's Encrypt)
- Monitoring (Prometheus + Grafana)

Main services:
✓ Keycloak (SSO/OIDC)
✓ PostgreSQL (Database)
✓ Redis (Cache)
✓ Traefik (Reverse Proxy)
✓ Taiga (Project Management)
✓ ProcessMaker (BPM)
✓ ERPNext (ERP)
✓ SuiteCRM (CRM)
✓ Nextcloud (Cloud Storage)
✓ FreeIPA (LDAP)
✓ Gitea (Git Repository)
✓ Mailcow (Email Server)
✓ MeshCentral (Remote Support)
✓ Prometheus (Metrics)
✓ Grafana (Visualization)
