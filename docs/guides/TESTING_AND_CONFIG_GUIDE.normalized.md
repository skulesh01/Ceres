# Тестирование и конфигурация (руководство)

Этот документ содержит шаги для тестирования и базовой конфигурации сервисов после развертывания.

## Быстрая проверка
1. Проверьте контейнеры:
```bash
docker compose ps -a
```
2. Просмотрите логи проблемных сервисов:
```bash
docker compose logs -f keycloak
```

## Настройка Keycloak
- Создайте realm `ceres` и клиентов для ProcessMaker, Taiga и Nextcloud.
- Проверьте редиректы и client secret.

## Настройка Nextcloud
- Создайте admin и app password для WebDAV.
- Проверьте доступ к `https://cloud.Ceres.local/remote.php/dav/files`.

## Проверки после конфигурации
- Убедитесь, что ключевые сервисы подключены (Keycloak ↔ ProcessMaker, Nextcloud ↔ ProcessMaker).
- Прогоните тестовый рабочий процесс в ProcessMaker.
