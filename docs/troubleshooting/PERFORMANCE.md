# Оптимизация производительности

## Рекомендации
- Увеличьте RAM и CPU для Docker Desktop (settings → Resources).
- Разрешите больше CPU/Memory для JVM-подобных сервисов.
- Используйте SSD для хранения Docker томов.
- В production разграничьте сервисы по нодам (k8s).

## Мониторинг
- Используйте Prometheus + Grafana для отслеживания метрик.
- Настройте алерты для загрузки CPU/RAM и свободного места на диске.
