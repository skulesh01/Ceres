# Проблемы с Docker и их решения

## 1. Docker Desktop не запускается
- Перезапустите Windows
- Убедитесь, что виртуализация включена
- Проверьте обновления Docker Desktop

## 2. Ошибка места на диске
- `docker system df` — проверьте использование
- `docker image prune -a` — удалить ненужные образы
- `docker volume ls` и `docker volume rm <name>` — осторожно с томами

## 3. Сеть контейнеров
- Проверьте `docker network ls`
- Удалите проблемную сеть и пересоздайте

## 4. Проблемы с правами файлов
- На Windows проверьте разрешения на папки, используемые томами
